# Changelog

## Corrections
- Correction de la persistance des paramètres Java et version Minecraft : les modifications sont maintenant sauvegardées et restaurées correctement.
- Mise à jour du label "Version Java requise" dans les paramètres pour se rafraîchir à la réouverture de la fenêtre et après détection automatique de Java.
- Synchronisation du label avec la version Minecraft sélectionnée via l'événement `change`.

- Affichage du changelog complet de la nouvelle version dans l'onglet "Mises à jour" des paramètres.
- Les notes de version (release notes) GitHub sont récupérées et affichées dans une boîte scrollable lors de la détection d'une mise à jour.
